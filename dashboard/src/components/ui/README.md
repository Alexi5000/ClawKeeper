# UI Components

This directory contains Shadcn UI components for the ClawKeeper dashboard.

## Currently Used Components

The following components are actively used in the application:

- `Badge.tsx` - Status badges and labels
- `Button.tsx` - Buttons and actions
- `Card.tsx` - Card containers for content
- `Input.tsx` - Form inputs
- `Modal.tsx` - Modal dialogs
- `Table.tsx` - Data tables
- `Skeleton.tsx` - Loading states
- `Tabs.tsx` - Tab navigation

## Available but Not Yet Used

The following standard Shadcn components are available for future use:

- `Alert.tsx` - Alert messages and notifications
- `Avatar.tsx` - User avatars
- `Calendar.tsx` - Date picker calendar
- `Dialog.tsx` - Alternative dialog component
- `DropdownMenu.tsx` - Dropdown menus
- `Form.tsx` - Form components with validation
- `Label.tsx` - Form labels
- `Popover.tsx` - Popover content
- `Progress.tsx` - Progress bars
- `ScrollArea.tsx` - Custom scroll areas
- `Select.tsx` - Select dropdowns
- `Separator.tsx` - Visual separators
- `Sheet.tsx` - Side panels
- `Toast.tsx` - Toast notifications
- `Tooltip.tsx` - Tooltips

These components are kept in the codebase as they are part of the standard Shadcn UI library and may be needed as features are expanded. They do not impact bundle size as they are only included when imported.

## Adding New Components

To add new Shadcn components, use the Shadcn CLI or copy component code from https://ui.shadcn.com
